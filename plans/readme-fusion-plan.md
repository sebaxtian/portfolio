## 🎨 Identidad Visual & UI/UX

Este repositorio no es solo código; es una extensión de mi identidad profesional. El diseño del portafolio web (`index.html`) sigue principios de:

- **Jerarquía Visual:** Uso de tipografías modernas (*Inter* para lectura, *JetBrains Mono* para acentos técnicos).
- **Paleta de Colores:** Un tema oscuro profundo (`#0f172a`) con acentos en Teal (`#14b8a6`) que evocan precisión y tecnología.
- **Interactividad:** Animaciones sutiles y transiciones suaves que mejoran la experiencia del usuario sin sacrificar el rendimiento.
- **Accesibilidad:** Estructura semántica HTML5 y optimización SEO integral.

## 🚀 Guía de Inicio Rápido

### Desarrollo Local
1. Clona el repositorio.
2. Abre `index.html` en tu navegador.
3. ¡Sin procesos de compilación! (Tailwind vía CDN).

### Servidor Local (Opcional)
```bash
# Python
python -m http.server 8000
# Node.js
npx serve
```

---

## 🔧 Personalización

- **Colores:** Edita la configuración de `tailwind.config` en el `<head>` de `index.html`.
- **Proyectos:** Duplica las tarjetas en la sección `#portfolio` y actualiza los datos.
- **SEO:** Modifica los meta tags y el JSON-LD estructurado al inicio del archivo.
